## 查看系统的path

echo $PATH

## 链接文件ln指令

格式：ln -option argv
进入链接目录执行，ln -s /xxx/*

## find查找文件的指令

格式：find path -option
查找文件系统文件，path查找的路径，选项用来制定查找的规则

## git基本命令

基本流程：
- git add *                 添加文件
- git commit -m "*"         提交文件
- git push origin master    推送文件到远端服务器

## 查看系统信息

命令：whoami   当前用户名
命令：uname    查看系统名
命令：hostname 查看主机名
